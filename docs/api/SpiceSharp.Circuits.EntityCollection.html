BuildOrderedComponentList()

This method is called when building an ordered list of circuit objects. Circuit objects will be called by descending priority.

Declaration
public void BuildOrderedComponentList()
Remarks

Ordering is necessary for entities that depend on one another. For example, a MutualInductance needs access to other Inductor in order to apply mutual coupling. So inductors need to come before mutual inductances.
